# bibliograph-storage-meadow

> Meadow DAL-backed storage provider for Bibliograph

- **Multi-Backend Persistence** -- SQLite, PostgreSQL, MySQL, MSSQL, and more
- **Full Bibliograph Interface** -- records, metadata, deltas, and key enumeration
- **Upsert Pattern** -- transparent create-or-update for all persist operations
- **Temporal Queries** -- enumerate records by timestamp range
- **Soft Delete** -- Meadow-managed audit trail for deletions
- **Auto-Schema** -- automatic table creation for SQLite
